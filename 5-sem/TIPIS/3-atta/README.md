## 3 аттестация
### Услевие из moodle
Необходимо придумать (или рассмотреть любое известное Вам устройство) модель системы с системой автоматического управления. Например, у Вас есть космический аппарат (не важно, спутник, ровер, зонд) с различными исполнительными механизмами (гусеницы, солнечные батареи, аппарат отбора проб, взрывотехническое оборудование ...). Необходимо разработать цепь управления исполнительным механизмом с учётом расстояния и специфики использования. В решении требуется привести модель системы и применить полученные на лекциях знания для оценки её характеристик.
